## 1. Description 
This is a simple Account API that allow users to manage account.

## 2. API Spec 
API spec openapi.json was generated with OpenAPI and additionnal changes. 

## 3. Tests
Rudamentary micro unit test built with pytest. Run test:
    - Install: pytest
    - Navigate to: app/
    - Execute: pytest
